# Bluff Dice Rules

이 문서는 현재 구현 기준의 게임 룰을 정리한다.

## 선언

- 일반 선언은 눈 `1`부터 `5`까지만 가능하다.
- 일반 선언 판정에서는 선언한 눈과 `6`을 함께 센다. 즉 `6`은 와일드다.
- `6`은 일반 선언 눈으로 선택할 수 없다.
- `6` 선언은 빨간 특수 칸에서만 가능하며, 판정할 때 진짜 `6`만 센다.
- 화면에서는 게임의 재미를 위해 `6`을 숫자 대신 별로 표시한다.

## 선언 순서

- 같은 수량에서는 눈 순서가 `1 -> 2 -> 3 -> 4 -> 5`로 증가한다.
- 일반 선언 2개급마다 `6` 특수 칸이 하나 열린다.
- 특수 칸 기준:
  - 일반 2개 뒤: `6` 특수 1개
  - 일반 4개 뒤: `6` 특수 2개
  - 일반 6개 뒤: `6` 특수 3개
  - 일반 8개 뒤: `6` 특수 4개
- 선언 맵 상한은 현재 생존자 총 주사위 수에 생존 인원 보너스를 더한 값이다.
  - 생존자 2~3명: 보너스 없음
  - 생존자 4~6명: 4명부터 1명당 2칸 추가
  - 생존자 7명 이상: 6명 보너스에 1명당 2칸 추가
  - 보너스는 최대 16칸이다.
- 따라서 인원이 많을수록 실제 총 주사위 수보다 조금 높은 선언까지 가능하다.

## 블러프

- 현재 선언자가 아닌 생존자는 누구나 블러프를 선언할 수 있다.
- 동시에 여러 명이 블러프를 누르면 서버에 먼저 도착한 하나만 인정한다.
- RPC는 라운드의 `challenge_locked`를 이용해 중복 정산을 막는다.

## 판정과 패널티

- 제시 개수보다 실제 개수가 적으면 블러프 성공이다.
- 블러프 성공 시 현재 선언자가 차이만큼 주사위를 잃는다.
- 제시 개수보다 실제 개수가 많으면 블러프 실패다.
- 블러프 실패 시 블러프한 사람이 차이만큼 주사위를 잃는다.
- 제시 개수와 실제 개수가 같으면 아무도 주사위를 잃지 않는다.
- 가진 주사위보다 많이 잃어야 하면 전부 잃고 탈락한다.

## 새 라운드

- 첫 라운드 시작자는 랜덤이다.
- 게임 시작 시 전체 주사위 최대치는 40개다.
- 각 플레이어의 시작 주사위는 `floor(40 / 시작 인원수)`개다.
- 나누고 남는 주사위는 버린다.
- 모든 새 라운드는 생존자의 주사위를 다시 굴린다.
- 블러프가 정확히 맞은 경우 블러프한 사람이 다음 라운드를 시작한다.
- 패널티를 받은 사람이 생존하면 그 사람이 다음 라운드를 시작한다.
- 다음 시작자가 탈락했다면 그 사람 다음 순서의 생존자가 시작한다.

## 방 종료

- 방 만들기, 참가, 시작, 선언, 블러프, 나가기 같은 명시적 동작이 있으면 방 활동 시간이 갱신된다.
- 상태 조회와 자동 새로고침은 활동으로 보지 않는다.
- 마지막 명시적 동작 이후 30분이 지나면 방은 승자 없이 종료된다.
